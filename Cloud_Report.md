Introduction

Cloud Computing is a technology that allows users to access computing resources such as servers, storage, databases, networking, and software through the internet. Instead of purchasing and maintaining physical hardware, users can use cloud services on demand. Cloud computing provides flexibility, scalability, cost efficiency, and easy access to resources from anywhere.

What is Cloud Computing?

Cloud Computing is the delivery of computing services over the internet. These services include storage, databases, networking, software, analytics, and computing power. Cloud computing helps organizations reduce infrastructure costs and improve efficiency.

Advantages of Cloud Computing
Cost Effective
High Scalability
Easy Accessibility
Improved Security
Automatic Updates
Better Resource Management
Cloud Service Models

Cloud services are mainly divided into three categories:

1. Infrastructure as a Service (IaaS)

Infrastructure as a Service provides virtualized computing resources over the internet. Users can access servers, storage, and networking resources without purchasing physical hardware.

Features
Virtual Machines
Storage Services
Networking Services
Flexible Resource Allocation
Examples
Amazon EC2
Microsoft Azure Virtual Machines
Google Compute Engine
Benefits
Reduced hardware costs
Easy scalability
Better flexibility
2. Platform as a Service (PaaS)

Platform as a Service provides a platform where developers can build, test, and deploy applications without managing the underlying infrastructure.

Features
Application Development Environment
Database Integration
Development Tools
Automated Deployment
Examples
Google App Engine
Microsoft Azure App Services
Heroku
Benefits
Faster application development
Reduced maintenance effort
Increased productivity
3. Software as a Service (SaaS)

Software as a Service provides ready-to-use software applications through the internet. Users can access these applications using a web browser without installing them locally.

Features
Web-based Access
Automatic Updates
Subscription-Based Usage
Easy Collaboration
Examples
Gmail
Google Drive
Microsoft 365
Benefits
Easy to use
No installation required
Accessible from anywhere
GitHub Deployment

For this project, GitHub was used to host a simple web page.

Steps Followed
Created a GitHub account.
Created a public repository.
Added an HTML file containing project information.
Enabled GitHub Pages.
Generated a public website URL.

GitHub Pages allows users to host static websites directly from a GitHub repository free of cost.

Conclusion

Cloud Computing has transformed the way organizations and individuals use technology. The three major cloud service models, IaaS, PaaS, and SaaS, provide different levels of services based on user requirements. Cloud computing offers flexibility, scalability, cost savings, and easy access to resources. GitHub Pages provides a simple and free method to deploy and host web applications online.
